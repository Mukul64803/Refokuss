# Refokuss Website Clone

This project is a clone of the Refokuss website, developed using React and Tailwind CSS. The website also features animations and transitions powered by the Framer Motion library, offering a smooth and visually engaging user experience.

## Features

- **Responsive Design**: Adapts to various screen sizes for a seamless user experience.
- **Animations**: Implemented with Framer Motion for smooth and fluid interactions.
- **Modular Components**: Built with reusable and modular React components.
- **Tailwind CSS**: Styled using utility-first Tailwind CSS for rapid UI development.

## Technologies Used
-  **React**: A JavaScript library for building user interfaces.
-  **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
-  **Framer Motion**: A React animation library for creating smooth and performant animations.
